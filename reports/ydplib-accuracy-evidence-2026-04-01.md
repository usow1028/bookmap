# Bookmap Accuracy Evidence Report (2026-04-01)

## 목적

`대림동1차한신아파트`를 기준 위치로 두고, 50권의 도서를 대상으로 Bookmap이 표시하는 영등포구 도서관 목록과 영등포구립도서관 홈페이지(`ydplib.or.kr`)의 실제 검색 결과가 얼마나 일치하는지 근거와 함께 정리한다.

이번 평가는 `2026-04-01`에 실행했다.

## 결론

- 위치: `대림동1차한신아파트 (37.494478, 126.903501)`
- 평가 범위: `영등포구 / ydplib.or.kr`
- 표본 도서 수: `50권`
- 표시 분관 수 합계: `175`
- 홈페이지 분관 수 합계: `255`
- 일치 분관 수: `47`
- 분관 기준 precision: `26.86%`
- 분관 기준 recall: `18.43%`
- 도서 기준 exact match: `3 / 50 = 6.00%`

해석:

- 현재 표시 결과와 실제 홈페이지 검색 결과는 높은 수준으로 일치하지 않는다.
- 특히 “표시 목록에는 있는데 홈페이지 검색에는 안 보이는 분관”과 “홈페이지 검색에는 나오지만 표시 목록에는 없는 분관”이 동시에 많이 발생했다.
- 따라서 현재 결과를 “실제 홈페이지 기준 정확한 보유/대출 상태”로 신뢰하기는 어렵다.

## 평가 기준

### 표시 결과

Bookmap의 원래 라이브 경로는 `libSrchByBook`를 사용하지만, 이번 실행 시점에는 정보나루 Open API가 차단되었다. 그래서 표시 결과는 대체 근거로 `data4library.kr` 공개 웹 상세페이지의 `bookOwnLibJson` 응답을 사용했다.

- 지역 코드: `11190` (`영등포구`)
- 데이터 의미: 해당 도서의 영등포구 소장 도서관 목록

### 실제 결과

실제 결과는 영등포구립도서관 통합 검색 페이지의 HTML을 직접 조회해 같은 ISBN을 가진 항목만 추출했다.

- 검색 엔드포인트: `https://www.ydplib.or.kr/intro/plusSearchResultList.do`
- 사용 필드: `searchType=SIMPLE`, `searchCategory=ALL`, `searchKey=ALL`, `searchLibrary=ALL`
- 파싱 항목: `도서관명`, `ISBN`, `대출가능/대출불가`, `예약 가능 여부`

### 지표 정의

- `precision`: 표시 분관 중 홈페이지에서도 확인된 비율
- `recall`: 홈페이지 분관 중 표시 결과에도 포함된 비율
- `exact match`: 한 권의 책에 대해 표시 분관 집합과 홈페이지 분관 집합이 완전히 동일한 경우

## 실행 근거

### 근거 1. 원래 Open API 경로는 차단됨

다음 오류가 실제로 재현되었다.

```text
Error: 1일 500건 이상 요청 시 IP 등록이 필요합니다. 등록 된 IP를 확인하시기 바랍니다.
```

즉 `libSrchByBook` 기반의 원래 경로는 이번 실행에서 그대로 평가에 사용할 수 없었다.

### 근거 2. 공개 웹 `bookOwnLibJson`은 영등포구 소장 도서관 목록을 반환함

`불편한 편의점`(`seq=5962217`)에 대해 공개 웹의 `bookOwnLibJson`을 영등포구(`11190`)로 조회했을 때 실제 응답 일부는 다음과 같았다.

이 응답은 “표시 결과”의 근거로 사용했다.

### 근거 3. 영등포구립도서관 홈페이지 검색은 같은 ISBN에 대해 다른 분관/상태를 보여줌

`불편한 편의점` 검색 결과 HTML에서 실제로 추출된 스니펫 일부:

```html
<span>ISBN: 9791161571188</span>
...
<span>도서관:
  대림1동 작은도서관
</span>
...
<b class="emp8">
대출가능[비치중]</b>
...
<span class="state typeA"><span class="ico"></span> 도서예약불가</span>
```

같은 책에 대해 홈페이지 기준으로 확인된 분관은 아래 2곳이었다.

- `여의도브라이튼도서관: 대출불가[비치중] / 도서예약불가`
- `대림1동 작은도서관: 대출가능[비치중] / 도서예약불가`

반면 표시 결과 쪽 영등포구 소장 목록은 아래 7개였다.

- 원지공원도서관
- 신길도서관
- 대림도서관
- 문래도서관
- 영등포생각공장도서관
- 여의샛강도서관
- 선유도서관

즉 이 책은 `표시 7 / 홈페이지 2 / 일치 0`이었다.

### 근거 4. exact match 사례도 실제로 존재함

`(철학하는 어린이) 자유가 뭐예요?`(`9788996029991`)는 표시 결과와 홈페이지 결과가 완전히 일치한 대표 사례였다.

표시 분관:

- 대림도서관
- 문래도서관
- 선유도서관

홈페이지 분관 및 상태:

- `대림도서관: 대출불가[대출중] / 도서예약신청`
- `문래도서관: 대출가능[비치중] / 도서예약불가`
- `선유도서관: 대출가능[비치중] / 도서예약불가`

따라서 이 책은 `표시 3 / 홈페이지 3 / 일치 3 / exact=Y`였다.

## 수치 해석

`precision 26.86%`는 표시된 분관 4곳 중 약 1곳만 홈페이지에서도 확인된다는 뜻이다.

`recall 18.43%`는 홈페이지에서 확인되는 분관 5곳 중 약 1곳만 표시 결과에도 들어온다는 뜻이다.

`exact match 6.00%`는 50권 중 47권에서 분관 집합이 완전히 달랐다는 의미다.

즉 현재 결과는 “대체로 맞는다”가 아니라 “분관 집합 자체가 자주 바뀐다” 쪽에 가깝다.
